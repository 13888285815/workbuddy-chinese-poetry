# 意念诗词库 Web UI

一个基于 Web 的中华古典诗词浏览工具，支持搜索、收藏、随机浏览等功能。

## 功能特性

- 📚 **树形目录** — 诗经、楚辞、唐诗、宋词、元曲等分类清晰
- 🔍 **全文搜索** — 支持按标题、作者、内容关键词搜索
- ⭐ **收藏功能** — 标记喜欢的诗词，随时回顾
- 🎲 **随机浏览** — 发现未知的经典之作
- 🌓 **主题切换** — 亮色/暗色模式
- 📱 **全屏阅读** — 沉浸式浏览体验
- 🔤 **字号调节** — 自定义阅读字号
- 💾 **零依赖** — 单文件 HTML，无需安装任何软件

## 诗词收录

| 分类 | 内容 |
|------|------|

| 其他 | 纳兰性德、幽梦影、御定全唐诗 |

## 使用方式

直接打开 `index.html` 即可使用，无需服务器。也可部署到 GitHub Pages 在线访问。

**本地预览：** 双击 `index.html`

**在线演示：** https://13888285815.github.io/workbuddy-chinese-poetry/

## 技术架构

- 单文件 HTML（内嵌 CSS + JavaScript）
- 树形数据内嵌，无需额外加载
- 诗文数据以 JSON 文件形式存储，按需加载
- 兼容 `file://` 协议和 HTTPS 部署
